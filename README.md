# TryHackMe Journey — Edyson Degand

Suivi de progression des machines TryHackMe : writeups, notes, scripts.

## Structure
- `/rooms/` : writeups organisés par date / machine
- `/notes/` : fiches techniques et cheatsheets
- `/playbooks/` : procédures réutilisables (enum, priv-esc, web)
- `/tools/` : scripts personnels
- `/screenshots/` : captures d'écran (pas d'infos sensibles)

## Règles importantes
- Pas de secrets ou de fichiers de configuration sensibles (`*.ovpn`, `*.pem`) dans le repo.
- Documente chaque étape dans `writeup.md` pour chaque room.
- Exemples de commit : `rooms(web-101): add writeup - enum and exploit`
